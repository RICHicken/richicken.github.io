Crawl Minecraft Plugin

Minecraft Crawl Plugin

Programming Language: Java

Libraries Used: SpigotMC

Github Link: github.com/RICHicken/MinecraftCrawl


Description:

This is a plugin for Minecraft: Java Edition (version 1.15.2) which allows players to take control of almost any hostile mob in the game! If a player dies, then the person who killed them becomes the main player. First one to beat the game is the player character wins!


Notes:

  • Allows players to control normally uncontrollable base-game monsters
  • Persists state of user-preferences across multiple runs of the server
