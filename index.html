Sudoku Puzzle Generator

April 2021

Programming Language: C#

Libraries Used: SFML

Github Repository - GUI Version: github.com/RICHicken/Sudoku-Puzzle-Generator

Github Repository - LaTeX Version: github.com/RICHicken/Sudoku-Generator-Latex


Description:

Summary:

Teacher Toolkit

August 2021

Programming Language: C#

Framework: WinForms

Download: Teacher Toolkit Release 1.0


Description:

Teacher Toolkit is a simple program made for some of teachers at a local middle school near me. The program is primarily focused on letting teachers randomizing students in some way. This includes creating raffles, which can be saved to ensure every student gets picked at least once. Assigning random students to jobs each day, making sure every student gets to be each job at least once. And creating student groups. This program allows teachers to do these tasks instantly since raffles and jobs can be saved, along with the student names.


Summary:

  • Allows teachers to add students to the program
  • Teachers can make raffles, which ensures every student is picked at least once
  • Teachers can distributes students to in-class jobs
  • Has a simple random student picker
  • Has a simple group creator
  • Raffles and Jobs can be saved and loaded

Secure TCP Chat

May 2023

Programming Language: Go

Github Repository: https://github.com/RICHicken/Secure_TCP_Chat


Description:

This project has a server and a client. The server will host a chat room for clients to join. Clients can join the server by inputting an IP address. Once a client joins, they can type their name and write to everyone else on the server. On top of that, all messages are encrypted using RSA encryption, so only connected members can see the messages.


Summary:

  • Built a server program that serves as a chat room.
  • Built a client that can to connect to servers and chat with other connected clients.
  • All messages are encrypted using RSA encryption.
